# Azure Storage Lister

Azure Function to list files in a Blob Storage container.

## Deployment Status
[![Deploy to Azure Function App](https://github.com/ajayagr/azure-storage-lister/actions/workflows/deploy.yml/badge.svg)](https://github.com/ajayagr/azure-storage-lister/actions/workflows/deploy.yml)

## Usage
`GET /api/list_files?container=<container_name>`
